# Proof of concept - Supertest

## Description

This project aims to display a practical usage of Supertest for Rest API testing. It shoots the request, gets the response and assert its content and status code. 

## Test Target (Serverest)

Serverest is a free study rest API that can be run locally. It gives us a commerce structure like a simple show with auth, users, products and shopping cart.
